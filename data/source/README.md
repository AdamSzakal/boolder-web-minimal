# Source Data

This directory contains the file-backed dataset for the static Boolder site.

## Layout

- `content/` — JSON metadata (areas, problems, circuits, clusters, pois, poi_routes, topos, lines)
- `media/topos/` — topo photos: `area-<id>/topo-<id>.jpg`
- `media/area-covers/` — area cover images: `area-cover-<id>.jpg`
